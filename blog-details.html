Autorizacion y Autenticacion

by Admin
Category: Tecnologia, Seguridad

En este post intentaremos cubrir de una forma práctica los elementos fundamentales para securizar nuestras aplicaciones web y APIs REST en .NET Core. Hablaremos de esquemas de autenticación, como funcionan, como usarlos y veremos que herramientas tenemos en .NET Core para autorizar de forma efectiva a nuestros usuarios.

¿Qué es la seguridad de las aplicaciones web?

Se refiere a las medidas de seguridad, a nivel de aplicación, cuyo propósito es impedir el robo o el secuestro de datos o códigos dentro de la aplicación.

¿Por qué es tan importante la seguridad de las aplicaciones?

La seguridad de las aplicaciones es importante porque las aplicaciones actuales suelen estar disponibles a través de varias redes, lo que aumenta las vulnerabilidades a los peligros y las amenazas a la seguridad.
